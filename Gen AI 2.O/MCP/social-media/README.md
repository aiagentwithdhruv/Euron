# Social Media MCP Server (YouTube + Instagram + Facebook)

MCP server for managing YouTube, Instagram, and Facebook from any MCP-compatible client. Uses YouTube Data API v3 with OAuth 2.0 and Meta Graph API for Instagram/Facebook.

## Tools

| # | Platform  | Tool                      | Description                                      |
|---|-----------|---------------------------|--------------------------------------------------|
| 1 | YouTube   | `youtube_search`          | Search YouTube videos by query                   |
| 2 | YouTube   | `youtube_channel_stats`   | Get channel statistics (subs, views, video count)|
| 3 | YouTube   | `youtube_video_details`   | Get detailed info for a specific video           |
| 4 | YouTube   | `youtube_my_videos`       | List authenticated user's uploaded videos        |
| 5 | Instagram | `instagram_profile`       | Get business profile info (followers, bio, etc.) |
| 6 | Instagram | `instagram_recent_posts`  | Get recent posts with likes, comments, media URL |
| 7 | Instagram | `instagram_post_insights` | Get insights for a specific post                 |
| 8 | Facebook  | `facebook_page_info`      | Get page info (name, followers, likes, category) |
| 9 | Facebook  | `facebook_recent_posts`   | Get recent page posts with engagement metrics    |
|10 | Facebook  | `facebook_post_to_page`   | Post a message to the Facebook page              |

## Quick Start

```bash
cd social-media
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### YouTube OAuth Setup

1. Create a project in [Google Cloud Console](https://console.cloud.google.com/)
2. Enable the **YouTube Data API v3**
3. Create OAuth 2.0 credentials (Desktop app) and download `credentials.json` into this folder
4. Run the auth flow:

```bash
python authenticate.py
```

This generates `token.json` which the server uses automatically.

### Instagram & Facebook Setup

1. Create a [Meta Developer App](https://developers.facebook.com/)
2. Connect your Instagram Business/Creator account to a Facebook Page
3. Generate a long-lived Page Access Token with the required permissions
4. Set the environment variables (see config below)

> **Note:** Instagram tools require a Business or Creator account connected to a Facebook Page.
> **Note:** Facebook tools require a Page Access Token with `pages_manage_posts`, `pages_read_engagement`, and `pages_show_list` permissions.

## MCP Client Configuration

```json
"SocialMedia": {
  "command": "/path/to/venv/bin/python",
  "args": ["/path/to/social_mcp.py"],
  "env": {
    "META_ACCESS_TOKEN": "your-long-lived-meta-access-token",
    "INSTAGRAM_BUSINESS_ID": "your-ig-business-id",
    "FACEBOOK_PAGE_ID": "your-fb-page-id"
  }
}
```

YouTube authentication is handled via `token.json` (generated by `authenticate.py`), so no additional env vars are needed for YouTube.
